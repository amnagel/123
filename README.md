# Система автоматизированного преобразования координат (Практическая №10)

Веб-сервис для автоматизированного пересчета пространственных координат (X, Y, Z) из одной системы в другую с использованием параметров трансформации и генерацией отчетной документации в формате MS Word.

## Ссылки на проект
* Интерфейс (Streamlit): https://ccrjdexg3hsrfanffj9wfw.streamlit.app/
* Бэкенд API (Render): https://one23-4bfu.onrender.com

## Технологический стек
* Frontend: Streamlit (Python) — создание интерактивного интерфейса.
* Backend: FastAPI (Python) — обработка данных и математические вычисления.
* Математика: Pandas (анализ данных), SymPy (рендеринг формул LaTeX), NumPy.
* Отчетность: Pandoc (pypandoc-binary) — динамическая генерация документов .docx из Markdown.

## Математический аппарат
Преобразование выполняется по модели подобия (7-параметрическое преобразование Гельмерта в упрощенном виде):

$$
\begin{pmatrix} X \\ Y \\ Z \end{pmatrix}_{new} = (1 + m \cdot 10^{-6}) \begin{pmatrix} 1 & \omega_z & -\omega_y \\ -\omega_z & 1 & \omega_x \\ \omega_y & -\omega_x & 1 \end{pmatrix} \begin{pmatrix} X \\ Y \\ Z \end{pmatrix}_{old} + \begin{pmatrix} dx \\ dy \\ dz \end{pmatrix}
$$

В текущей версии интерфейса реализованы параметры линейного сдвига (dx, dy, dz) и масштабный коэффициент (m).

## Инструкция пользователя
1. Загрузка данных: Подготовьте Excel или CSV файл, содержащий колонки X, Y, Z. Система автоматически находит строку с заголовками в пределах первых 10 строк файла.
2. Ввод параметров: В боковой панели (Sidebar) введите значения смещений dx, dy, dz и масштабный коэффициент m.
3. Расчет: Нажмите кнопку "Расчет". Данные будут переданы на сервер Render для выполнения вычислений.
4. Результат: После завершения обработки отобразится таблица с результатами и станет доступна кнопка "Скачать DOCX" для получения отчетного документа.

## Структура репозитория
* main.py — серверная часть на FastAPI (обработка файлов, логика трансформации, генерация Word).
* app.py — клиентская часть на Streamlit (интерфейс, взаимодействие с API).
* req.txt — список необходимых зависимостей для сборки проекта в облачных сервисах.
