# Система трансформации пространственных координат (Coordinate Transformation System)

Веб-сервис для автоматизированного пересчета координат (X, Y, Z) между различными системами отсчета. Проект реализован в рамках клиент-серверной архитектуры с использованием современных веб-фреймворков для научных вычислений.

## Технологический стек
* Frontend: Streamlit (Python) — обеспечение интерактивного ввода данных и визуализации результатов.
* Backend: FastAPI (Python) — высокопроизводительный сервер для выполнения математических операций.
* Обработка данных: Pandas (анализ табличных данных), Openpyxl (поддержка формата Excel).
* Математический аппарат: SymPy (обработка символьных вычислений и LaTeX), NumPy.
* Генерация документации: Pandoc (pypandoc-binary) — автоматизированная сборка отчетов в формате .docx.

## Математический аппарат
Преобразование координат выполняется по модели подобия (преобразование Гельмерта) в матричной форме:

$$
\begin{pmatrix}
X \\
Y \\
Z
\end{pmatrix}_{new}
=
(1 + m \cdot 10^{-6})
\begin{pmatrix}
1 & \omega_z & -\omega_y \\
-\omega_z & 1 & \omega_x \\
\omega_y & -\omega_x & 1
\end{pmatrix}
\begin{pmatrix}
X \\
Y \\
Z
\end{pmatrix}_{old}
+
\begin{pmatrix}
dx \\
dy \\
dz
\end{pmatrix}
$$
В текущей конфигурации системы доступны параметры линейного смещения (dx, dy, dz) и масштабный коэффициент (m).

## Инструкция пользователя
1. Подготовка данных: Загрузите файл формата Excel или CSV. Система автоматически идентифицирует столбцы X, Y, Z, проводя поиск заголовков в первой части документа.
2. Ввод параметров трансформации: В боковой панели интерфейса укажите значения для смещения осей и масштабирования.
3. Запуск вычислений: Нажмите кнопку "Расчет". Процесс включает передачу данных на удаленный сервер Render, выполнение математических преобразований и формирование Markdown-отчета.
4. Экспорт результатов: По завершении обработки доступна таблица с результатами и кнопка для скачивания сформированного документа в формате MS Word.

## Структура репозитория
* main.py — реализация серверной части на FastAPI (обработка запросов, алгоритмы трансформации, конвертация в Word).
* app.py — реализация клиентской части на Streamlit (управление интерфейсом, взаимодействие с API).
* requirements.txt — перечень программных зависимостей для сборки и развертывания проекта.
